L3 Reviewer — Page Flow Approaches

Three ways to structure the input and output experience. Which feels right?

A

Step-by-step Wizard recommended

A multi-step form guides the assessor through each section in order:

  1. Admin fields (Sections 1–3) + transcript upload
  2. Planning & Preparation (Section 4)
  3. Self-reflection (Section 9)
  4. Submit → Results screen with download buttons

Pros

  • Not overwhelming
  • Clear progress
  • Mirrors the paper form

Cons

  • More clicks to get to results
B

Single Scrolling Page

All inputs on one long page — admin fields, text areas, transcript upload — then a "Review" button at the bottom. Results replace the form.

Pros

  • Everything visible at once
  • Familiar form layout

Cons

  • Long page can feel daunting
  • Easy to miss fields
C

Upload-first, Context-second

Transcript upload is the hero action. Admin fields and text inputs are collapsed in an optional "Add context" accordion — assessor can skip them and just get the AI review.

Pros

  • Fastest path to results
  • Context is optional/non-blocking

Cons

  • Users may skip important context
  • Report has gaps if fields left blank
